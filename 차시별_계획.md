# 🧭 기하 탐험대: 평면도형의 이동 — 8차시 상세 계획

> 원본: `Geometry_Quest_Plan.md` (게이미피케이션 8차시 가이드)
> 정리 방향: 묶음 차시(3~4, 5~6, 7~8)를 **개별 차시로 분리**, 각 차시마다 **도입·전개·정리 / 산출물 / 평가 / 도구**를 구체화.

---

## 🗺️ 전체 시나리오
- **세계관:** 고대 기하 대륙의 유물이 사악한 마법사에 의해 흩어짐. 학생들은 '기하 탐험대'가 되어 차시별 미션 클리어.
- **보상:** 차시당 '유물 조각' 1개 → 8개 모으면 최종 칭호 **"기하 마스터"**.
- **공통 루틴:** 도입(스토리 5분) → 전개(미션 25분) → 정리(개념 정리 10분).

## 1차시 — [밀기①] 화물 운송 대작전
| 항목 | 내용 |
|---|---|
| **학습 목표** | 도형을 상·하·좌·우로 밀었을 때 **모양·크기는 그대로, 위치만 변함**을 이해. |
| **핵심 개념** | 평행이동(Translation) |
| **도입** | 스토리 — "마법사가 화물(도형)을 흩뿌렸다. 정해진 칸으로 운송하라!" |
| **전개 활동** | **도형 소코반** — 모눈종이 위 도형을 지정 칸까지 N칸 밀기. 장애물 회피. |
| **정리/핵심 질문** | "오른쪽으로 5칸 밀면 왼쪽 꼭짓점은 어디로?" |
| **산출물** | 모눈종이 운송 미션지(3문제) |
| **평가** | 자기 평가(이해도 ★1~5) + 미션 정답률 |
| **도구** | 모눈종이, 도형 블록 / Mathigon Polypad |
| **유물 조각** | 🟦 1번 |

## 2차시 — [뒤집기①] 거울의 성 탈출
| 항목 | 내용 |
|---|---|
| **학습 목표** | 도형을 좌·우·위·아래로 뒤집었을 때의 모양 변화를 이해. |
| **핵심 개념** | 대칭이동(Reflection) — 뒤집는 방향 = 대칭축 |
| **도입** | 스토리 — "거울의 성에 갇혔다. 거울을 통과한 도형의 모습을 그려라!" |
| **전개 활동** | **데칼코마니 암호 해독** — 절반만 그려진 도형 → 4방향 뒤집기 결과 추측. 기름종이로 검증. |
| **정리/핵심 원리** | 좌우 뒤집기는 **왼쪽↔오른쪽**, 상하 뒤집기는 **위↔아래** 교환. |
| **산출물** | 4방향 뒤집기 그림판(완성본) |
| **평가** | 짝과 답 비교 → 일치율 기록 |
| **도구** | 기름종이, 사인펜, 거울 |
| **유물 조각** | 🪞 2번 |

## 3차시 — [돌리기①] 비밀 금고의 회전 다이얼 (90°·180°)
| 항목 | 내용 |
|---|---|
| **학습 목표** | 도형을 시계/반시계 방향으로 **90°, 180°** 돌렸을 때의 변화를 이해. |
| **핵심 개념** | 회전이동(Rotation) — 회전 중심·각도 |
| **도입** | 스토리 — "금고 다이얼을 정확히 맞춰야 유물 보관함이 열린다." |
| **전개 활동** | **다이얼 자물쇠 해제 (기초)** — 도형을 90°·180° 돌려 자물쇠 구멍에 맞추기. 회전판 교구 활용. |
| **정리/핵심 질문** | "90° 시계방향과 90° 반시계방향의 결과는 같을까?" |
| **산출물** | 회전 결과 스케치(90°/180°×시계·반시계 = 4장) |
| **평가** | 회전 결과 매칭 퀴즈 |
| **도구** | 회전판(실물), 모눈종이 / Mathigon Polypad |
| **유물 조각** | 🔄 3번 |

## 4차시 — [돌리기②] 270° 돌리기와 회전의 규칙
| 항목 | 내용 |
|---|---|
| **학습 목표** | **270°** 회전 결과 추론, 90°와 270°의 상관관계 이해. |
| **핵심 개념** | 회전이동의 합성 — 시계 90° = 반시계 270° |
| **도입** | 스토리 — "고급 다이얼! 270°를 돌려야 마지막 자물쇠가 열린다." |
| **전개 활동** | **다이얼 자물쇠 해제 (심화)** — 270° 시계/반시계 회전 결과 예측 → 검증. 90° 회전을 3번 반복하면 270°와 같음을 발견. |
| **정리/미션** | "시계 270° = 반시계 ?°" 빈칸 채우기. |
| **산출물** | 회전각 변환표(0°·90°·180°·270°×시계/반시계) |
| **평가** | 동료 피드백 — 짝의 변환표 검토 |
| **도구** | 회전판, 각도기 |
| **유물 조각** | 🔃 4번 |

## 5차시 — [복합①] 미궁 탈출 — 두 동작 연결
| 항목 | 내용 |
|---|---|
| **학습 목표** | 밀기·뒤집기·돌리기 중 **2개 동작을 연속**으로 수행한 결과 추론. |
| **핵심 개념** | 변환의 합성(첫 단계) |
| **도입** | 스토리 — "미궁의 갈림길. 명령어 2개를 조합해 다음 방으로!" |
| **전개 활동** | **복합 명령 프로그래밍 (기초)** — 예: "밀기 3칸 → 90° 돌리기" 카드를 조합해 도형을 목적지로 이동. |
| **정리/핵심 질문** | "뒤집고 돌리기 vs 돌리고 뒤집기 — 결과가 같은가?" |
| **산출물** | 명령 카드 시퀀스(2동작) + 도착 위치 그림 |
| **평가** | 미션 클리어 시간 + 시도 횟수 |
| **도구** | 명령 카드(인쇄), 모눈종이 |
| **유물 조각** | 🧩 5번 |

## 6차시 — [복합②] 보스 챌린지 — 순서의 비밀

| **전개 활동** | **복합 명령 프로그래밍 (심화)** — 같은 동작 3개를 순서만 바꿔 비교. 보물 상자 정확히 도달하는 시퀀스 찾기. |

| **도구** | 명령 카드, Mathigon Polypad(시뮬레이션 검증) |

## 🏆 평가 체계 (8차시 통합)
1. **자기 평가** — 차시 종료 시 ★1~5 이해도 자기 기록.
2. **동료 피드백** — 5·7·8차시에서 짝/모둠 단위 상호 검토.
3. **수행 평가** — 8차시 산출물(디지털 테셀레이션 + 캡션)을 종합 평가.
4. **유물 조각 누적** — 8개 모두 획득 시 "기하 마스터" 인증.

## 📌 차시 흐름 요약
```
1차시 밀기 → 2차시 뒤집기 → 3차시 돌리기(90·180) → 4차시 돌리기(270·규칙)
        ↓
5차시 복합(2동작) → 6차시 복합(3동작·순서) → 7차시 테셀레이션 설계 → 8차시 전시·인증
```
