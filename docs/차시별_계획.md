# 🧭 기하 탐험대 — 평면도형의 이동 (8차시 마스터 플랜)

> **세계관**: 고대 기하 대륙의 유물이 사악한 마법사에 의해 흩어졌다. 학생들은 '기하 탐험대'가 되어 차시별 미션을 클리어하며 유물 조각을 수집한다. 8개 모두 모으면 **"기하 마스터"** 칭호를 획득.
>
> **공통 루틴**: 도입(스토리 5분) → 전개(미션 25분) → 정리(개념 정리 10분)
>
> **누적 보상**: 차시당 유물 조각 1개 → 8개 합체 = 마스터 인증

## 2차시 — [뒤집기①] 거울의 성 탈출 🚧
| 항목 | 내용 |
|---|---|
| **학습 목표** | 도형을 좌·우·위·아래로 뒤집었을 때의 모양 변화를 이해. **모양·크기는 그대로, 좌우(또는 상하)만 바뀜**을 발견. |
| **핵심 개념** | 대칭이동(Reflection) — 뒤집는 방향 = 대칭축 |
| **도입** | 스토리 — "거울의 성에 갇혔다. 마법 거울에 도형을 비춰 잠긴 문을 열어라!" |
| **전개 활동** | **데칼코마니 암호 해독** — 비대칭 도형(F자·물고기·깃발)을 좌우/상하 뒤집어 골 모양·위치 일치시키기. 두 번 뒤집으면 원래대로 돌아옴을 발견. |
| **정리/핵심 원리** | 좌우 뒤집기 = 왼쪽↔오른쪽 / 상하 뒤집기 = 위↔아래 / 같은 축 두 번 = 원래대로 / 좌우+상하 = 180° (3차시 복선) |
| **산출물** | 4방향 뒤집기 그림판(완성본) |
| **평가** | 짝과 답 비교 → 일치율 기록 |
| **도구** | 기름종이, 사인펜, 거울 / `mumu뒤집기` 게임 |
| **유물 조각** | 🪞 2번 |
| **게임 메커니즘** | 도형 4방향 이동 + ↔(좌우 뒤집기) / ↕(상하 뒤집기) · 비대칭 도형(F·G·L·S) |
| **스테이지** | 20개 / 5티어 (A 좌우뒤집 → B 상하뒤집 → C 데칼코마니 → D 합성 → E 마스터) |

## 3차시 — [돌리기①] 비밀 금고의 회전 다이얼 (90°·180°)
| 항목 | 내용 |
|---|---|
| **학습 목표** | 도형을 시계/반시계 방향으로 **90°, 180°** 돌렸을 때의 변화를 이해. |
| **핵심 개념** | 회전이동(Rotation) — 회전 중심·각도 |

| **전개 활동** | **다이얼 자물쇠 해제 (기초)** — 도형을 90°·180° 돌려 자물쇠 구멍에 맞추기. 회전판 교구 활용. |
| **정리/핵심 질문** | "90° 시계방향과 90° 반시계방향의 결과는 같을까?" |
| **유물 조각** | 🔄 3번 |

## 4차시 — [돌리기②] 270° 돌리기와 회전의 규칙
| 항목 | 내용 |

## 🏆 평가 체계 (8차시 통합)
1. **자기 평가** — 차시 종료 시 ★1~5 이해도 자기 기록.
2. **동료 피드백** — 5·7·8차시에서 짝/모둠 단위 상호 검토.
3. **수행 평가** — 8차시 산출물(디지털 테셀레이션 + 캡션)을 종합 평가.
4. **유물 조각 누적** — 8개 모두 획득 시 "기하 마스터" 인증.
